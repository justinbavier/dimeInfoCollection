Mission Statement

Our goal is to seamlessley incorporate charitable donations into people's daily lives. In doing so we hope to set the new standard for how charitable donations are collected. We believe making substantial charitable donations is an achievable goal through nominal daily contributions that add up over time.

Don't worry, we aren't going to spam you with a bunch of e-mails. We'll only hit you up twice, once when we begin beta testing and again when we launch!
